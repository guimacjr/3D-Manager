# Mockups - 3D Manager

Arquivo principal:
- `mockups/MockupApp.tsx`

Telas cobertas:
- Dashboard principal.
- Lista de impressoras.
- Cadastro de impressora.
- Lista de filamentos.
- Cadastro de filamento.
- Custos fixos.
- Lista de orcamentos.
- Novo orcamento.

Observacoes:
- Mockup visual e funcional de navegacao local.
- Sem persistencia em banco.
- Dados exibidos sao exemplos estaticos.

Uso rapido:
1. Copiar `MockupApp.tsx` para um projeto React Native/Expo.
2. Definir esse componente como tela inicial.
3. Rodar no emulador/dispositivo para validar fluxo e campos.
